# خيارات بطاقة المنتج
import ProductCardDemo from "assets/screenshots/theme-options/product-card.png"

<br/>
<Screenshot src={ProductCardDemo} alt="Product Card Demo" />

<br/>
يمنحك الثيم تحكم كامل في شكل شكل بطاقة المنتج من تحكم في الوان الخلفية الي لون العنوان , و الاطار , ايضا تحكم في اتجاه النص داخل بطاقة المنتج و امكانية اخفاء ايقونة السلة و زر المعاينة السريعة
<br/>

يمكنك تخصيص بطاقة المنتج باستخدام الخيارات التالية:
- امكانية جعل حواف بطاقة المنتج مستديرة
- امكانية اضافة اطار لبطاقة المنتج مع تحكم بلون الاطار
- امكانية تغير لون الخلفية لبطاقة المنتج
- امكانية تغير لون الخط لبطاقة المنتج
- امكانية تغير شكل ايقونة السلة
- امكانية اخفاء زر اضافة الى السلة
- امكانية زر المعاينة السريعة
- امكانية التحكم في اتجاه النصوص
